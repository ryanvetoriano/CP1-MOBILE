# 📱 Formulário de Cadastro - React Native

Aplicativo simples desenvolvido em React Native com um formulário de cadastro.

## 🚀 Funcionalidades

- Preenchimento dos campos:
  - Nome
  - Curso
  - Disciplina
  - Descrição
- Validação de campos obrigatórios
- Exibição dos dados após envio
- Limpeza do formulário após envio

## 🛠️ Tecnologias

- React Native
- Expo
- JavaScript

## ▶️ Como executar

```bash
npm install
npm start
```

## 📱 Como funciona

1. Preencha o formulário  
2. Clique em "Enviar"  
3. Os dados aparecerão na tela  
4. Se algum campo estiver vazio, será exibido um erro
